HTML

HTML is a markup language used to structure the content of web pages. It consists of a series of elements that tell web browsers how to display text, images, links, and other media. HTML provides a structured way to organize information and give it meaning, enabling browsers to interpret and present content correctly.

CSS

CSS, short for Cascading Style Sheets, is a style sheet language used to describe the look and formatting of a document written in HTML or XML. It provides a way to control the colors, fonts, spacing, and other visual aspects of a web page. By separating the presentation from the content, CSS enables web designers to create consistent and visually engaging websites.

javascript

JavaScript is a high-level programming language that allows you to add interactivity, perform calculations, manipulate web page elements, and create dynamic content on the web. Unlike HTML and CSS, which focus on structure and presentation, JavaScript focuses on behavior and functionality.

GitHub

GitHub is a web-based platform that provides version control and collaboration for developers working on software projects. It serves as a centralized repository where developers can store, manage, and collaborate on code. GitHub is built on top of Git, a distributed version control system.

React

React is a JavaScript library for building user interfaces. It provides a declarative and efficient way to create interactive and reusable UI components. React was developed by Facebook and has gained immense popularity due to its simplicity, performance, and component-based architecture.

VS Code

Visual Studio Code is a free and open-source code editor developed by Microsoft. It provides a powerful and customizable environment for writing, editing, and debugging code across various programming languages. VS Code is renowned for its simplicity, performance, and extensive ecosystem of extensions.
